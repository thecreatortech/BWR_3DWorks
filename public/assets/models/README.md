# 3D Model Assets

Place your GLB files in this folder. Each product should have a corresponding GLB file.

## File Naming Convention

- `orbital-vase.glb` - Orbital Vase No.7
- `helical-tower.glb` - Helical Tower
- `void-sphere.glb` - Void Sphere
- `lattice-cuff.glb` - Lattice Cuff
- `arc-plinth.glb` - Arc Plinth
- `moire-pendant.glb` - Moiré Pendant
- `fractal-bowl.glb` - Fractal Bowl
- `column-series.glb` - Column Series I
- `mesh-ring.glb` - Mesh Ring

## Quick Start

1. Generate images using the prompts above with your preferred AI tool
2. Convert images to 3D STL files using:
    - **Meshy.ai** (easiest, AI text/image to 3D)
    - **Rodin AI** (high quality)
    - Or use existing STL files from Thingiverse, Printables, etc.
3. Place STL files in this folder with the names listed above
4. The website will automatically load and display them!
